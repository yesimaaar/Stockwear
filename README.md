# StockWear - Sistema de Gestión de Inventario con reconocimiento visual de productos

# Instalar dependencias
npm install

En caso de error, utilizar:  
npm install --legacy-peer-deps

# Ejecutar en desarrollo
npm run dev

ahora es: 
npm install -g pnpm
pnpm install
pnpm run dev --webpack
